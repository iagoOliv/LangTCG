# LangTCG

LangTCG é um gerador de Cards customizados de linguagens de programação feito em React em desenvolvimento! (Sim, é isso que leu).

## Instalando

### Instale todas as dependências
```
npm i
```

### Rode no LocalHost
```
npm run dev
```

### Construa a página estática - (Opcional)
```
npm run build
```
